# API COM NODEJS

## Instalar:
```
npm i express sequelize mysql2 cors
```

## iniciar projeto:
```
npm init -y
```

## Codigo no Mysql
```
create database if not exists db_api;

use db_api;

show tables;

select * from clientes
```
